# Hissab-Kitab – Smart Inventory Management System

HissabKitab is a smart inventory management system designed to simplify tracking and managing stock. It allows users to efficiently add, update, and monitor inventory items while maintaining accurate records.

## 🚀 Features

* Add and manage inventory items
* Update stock details dynamically
* Real-time data storage using Firebase
* Simple and user-friendly interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Firebase
